# dot-dots

Basic dot files configuration

## Install
This dot files config is available as [Ellipsis][Ellipsis] package.

#### Ellipsis
``` shell
    ellipsis install sskytyba/dot-dots
```

## Maintain
To keep everything up-to-date;

#### Ellipsis
```shell
    ellipsis pull dots
```
